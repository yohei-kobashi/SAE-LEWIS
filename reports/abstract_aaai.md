## Abstract(日本語)

スパース自己符号化器(SAE)は、大規模言語モデル(LLM)の内部状態を解釈可能な活性の組に分解し、過去形や敬語といった言語学的特徴との対応づけを可能にする。しかし、特定された活性が本当にその特徴を担うかを確かめる因果検証の標準手段である活性への直接介入(定数への書き換えや単一方向の加算)は、効果が弱く不安定であることが先行研究により報告されている。また全てのトークンに均一に介入する方法が一般的であり複雑な特徴に厳密に対応した介入を行うことは難しい。本研究は、介入を固定ルールではなく学習された編集器に生成させる新手法(SAEスペック条件付きedit-flow介入)を提案する。LLM2Vecという手法で対象となるLLMをbi-directional encoderに変換した上でEdit Flowsを基盤とした編集モデルとして学習する。その編集モデルをさらに凍結LLMのresidual streamsに介入するモデルとして追加学習することでSAE活性への介入を受けてLLMの内部表現を編集するモデルとなる。提案手法は指定されたSAE活性（spec）と入力文を読み、編集ベクトルΔhを残差ストリームに注入する。テキストは凍結LLM自身が生成するため、最小対の編集成功が活性同定の因果的証拠となる。学習用データはDolma v1.6のサンプルから文を抽出した上でを分かち書き、形態素解析、統語解析を用いて確率的に書き換えた文'を作成し、最終的に22万のペアを学習した。評価では同じ文を復唱させるプロンプトを用いて行った。介入に成功すると1つの言語学的特徴だけ書き変わって元の文ではなくペアの文を復唱するというテストとなる。データにはLinguaLens-Dataを用い、形態論、統語論、意味論、語用論の4分類に属す英語99特徴の最小対編集を用いた。実験ではLinguaLens-Dataを9:1に分け、9を特徴に対応する活性の特定、1をテストに用いた。実験の結果、ペア文との完全一致評価ではpromptingには一部を除き及ばなかったものの、layer 12でenhancementで0.148、ablationで0.128とLinguaLensなどの既存介入手法を超える結果が得られた。そして、因果効果を測るFeature Intervention Confidence（FIC）では0.463とpromptingの0.410を上回る値を出し「同定された活性の因果検証」が学習された介入によって実用的な精度で可能になることを示した。

## Abstract (English translation, 2026-07-22)

Sparse autoencoders (SAEs) decompose the internal states of large language models (LLMs) into interpretable sets of activations, enabling their alignment with linguistic features such as past tense and honorifics. However, direct activation intervention — overwriting activations with constants or adding a single direction — the standard means of causally verifying that identified activations actually carry a feature, has been reported to be weak and unstable. Moreover, such interventions are typically applied uniformly across all tokens, making precise intervention on complex features difficult. We propose SAE-conditioned edit-flow intervention, which generates interventions with a learned editor rather than fixed rules. We convert the target LLM into a bidirectional encoder via LLM2Vec and train it as an editing model based on EditFlows; we then further train it to intervene on the frozen LLM's residual stream, yielding a model that edits the LLM's internal representations in response to SAE-activation interventions. Our method reads a specified set of SAE activations (a spec) and the input sentence, and injects an edit vector Δh into the residual stream. Because the text is generated by the frozen LLM itself, successful minimal-pair editing constitutes causal evidence for activation identification. For training, we extracted sentences from a sample of Dolma v1.6 and stochastically rewrote them using sentence segmentation, morphological analysis, and syntactic parsing with masked-LM word proposals, yielding 220k training pairs. Evaluation uses a prompt that asks the frozen LLM to repeat the input sentence verbatim: under a successful intervention, exactly one linguistic feature changes, so the model repeats not the original sentence but its minimal-pair counterpart. We use LinguaLens-Data — 99 English features spanning morphology, syntax, semantics, and pragmatics — split 9:1 into an identification set for locating feature-aligned activations and a test set. In exact-match evaluation the proposed method falls short of prompting with some exceptions, yet surpasses existing intervention methods including LinguaLens. On Feature Intervention Confidence (FIC), which measures causal effect, it exceeds prompting, demonstrating that causal verification of identified activations becomes practical with a learned intervention.

## 英語版の骨子(本文執筆時の参照用)

1. Background: SAEs decompose LLM internals; linguistic-feature
   identification (LinguaLens lineage) needs causal verification.
2. Problem: direct activation interventions are weak/unstable (their own
   admission); identification itself is selection-unstable (our
   split-half: >40% top-1 turnover) and lacks data separation.
3. Method: a learned editor (Edit Flows backbone, LLM2Vec encoder) reads
   an offline per-feature spec (signed pool-mean over held-out
   identification pairs; stability 0.83) + the input, and emits Δh into
   the frozen LM's residual stream; text comes from the frozen LM.
4. Training: self-supervised corruption-restoration on a general corpus
   only (zero-shot w.r.t. the evaluation set).
5. Results: best exact among interventions with unique specificity
   (random ≈ 0); FIC (ablation) 0.85–0.94 across layers vs ≈0 for
   clamping; prompting wins raw exact but fails specificity.
6. Claim: causal verification of identified activations becomes
   practical only with a learned intervention generator.
